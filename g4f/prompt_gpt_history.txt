assistant: Для того чтобы нарисовать график биткоина на Python, можно использовать библиотеку `matplotlib` для построения графиков и библиотеку `yfinance` для получения данных о стоимости криптовалюты.

Ниже приведен пример кода, который рисует график цены биткоина за последние 30 дней:

### Установка необходимых библиотек:
Если у вас не установлены библиотеки `yfinance` и `matplotlib`, вы можете установить их с помощью pip:

```bash
pip install yfinance matplotlib
```

### Код для рисования графика:

```python
import yfinance as yf
import matplotlib.pyplot as plt

# Загружаем данные о цене биткоина (символ: BTC-USD) за последние 30 дней
btc_data = yf.download('BTC-USD', period='30d')

# Рисуем график цены закрытия (Close price)
plt.figure(figsize=(10, 6))
plt.plot(btc_data['Close'], label='Цена закрытия')
plt.title('График курса Биткоина за последние 30 дней')
plt.xlabel('Дата')
plt.ylabel('Цена в USD')
plt.grid(True)
plt.legend()

# Показываем график
plt.show()
```

### Описание кода:
1. **Загрузка данных**: Используется метод `yf.download()` из библиотеки `yfinance`, чтобы получить данные по биткоину с тикером "BTC-USD". Параметр `'period'` определяет временной интервал (в данном случае — последние 30 дней).
2. **Построение графика**: С помощью функции `plot()` из библиотеки `matplotlib`, мы строим линию, представляющую цену закрытия.
3. **Оформление**: Добавляем заголовок, метки осей и сетку для лучшего восприятия.
4. **Отображение**: Функция `show()` отображает окно с готовым графиком.

Этот код должен корректно работать на любой системе Windows при наличии установленных библиотек Python.
